Change log

UPDATE 16/02/2017 - Pensolve v3

  • Connected Google sheets add-on, can now process files to PDF from within the spreadsheet
  • Added OAuth2 flow to allow third party apps to connect to Pensolve
  • Improved display of flowmap and comparison view
  • Can now update public files

UPDATE 21/01/2017

  • Added auto detection to determine whether a master file has been modified
  • Improved display of flowmap and comparison view
  • Can now update public files
  • Added pricing page and monthly subscriptions
  • Can now pay online

UPDATE 30/11/2016

  • Added support for INDIRECT function and values in strings using the ampersand symbol
  • Improved output when referencing an empty cell
  • Support for dropdown items selected from cells
  • Outputs are now underlined in PDF
  • Recognition of repeated expression that is expanding (e.g. sum over an increasing range)
  • Better logic for piecewise functions (uses equal to rather than less than)
  • Allow piecewise functions in flowmap
  • Fixed issue where double space in spreadsheet name caused no PDF to be produced

UPDATE 22/10/2016

  • Added support to mark files as master files
  • Added a line break between summary text that has an empty cell vertically in between
  • Added support for symbol font for greek letters
  • Increased PDF compiling time - by tidying up underscore syntax

UPDATE 17/10/2016

  • Images in spreadsheet can be marked to show up in PDF
  • Added support for PENBREAK, a page break that splits your spreadsheet vertically Learn more here

UPDATE 05/09/2016

  • Fixed issue with how multiple notes were printed
  • Moved unused cells to end of PDF output
  • Added flow map view in beta
  • Tables can better handle long names and long text

UPDATE 29/08/2016

  • Hyperlinks in spreadsheets now show as hyperlinks in PDF
  • Long calc references now added to comments, instead of going off page
  • Fixed issue with google drive spreadsheets numbers were not rounding
  • Fixed issue with description/comments going on to a different page to the equations
  • Percentage is now recognised as a unit
  • Improved PDF output of COT function and other advanced excel functions
  • A "-" cannot be a comment

UPDATE 22/08/2016

  • Fixed issue where Pensolve used banker rounding, changed to high school rounding
  • Added support for units within cells
  • Fixed issue where summary text was not recognised if sheet contained unused cells

UPDATE 25/07/2016

  • Job number now shows up on title page of PDF
  • Fixed issue with subsection count
  • Fixed issue with white space in comment, summary text and issue with colon at end of text
  • Fixed issue with sub section header number not restarting
